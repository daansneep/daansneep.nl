React

I had started learning react after I had finished two big projects that were written in angular. There were advantages when starting to learn react because of this, but there were also some downsides. It was useful to have worked with a frontend framework before, it made me aware of possible problems that I could also encounter with react which made the learning process somewhat easier. But because angular was such a huge framework and relatively speaking react such a minimal framework, it turned out to be quite difficult to pick up. I eventually learned how to build a responsive webapp with react and mobx as a state manager.
